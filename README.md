## swoole+vue实现的web聊天
#####在线演示 [catteacher](catteacher.cn)
### 环境要求
* PHP >= 7.0
* Swoole4.2
* composer
* sqlite3
## 启动 server
php index.php start|reload|stop

## 前端
```
// 安装依赖
npm install
npm run dev
```
